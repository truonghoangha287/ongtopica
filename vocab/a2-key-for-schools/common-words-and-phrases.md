## Common Exam Phrases
| Phrase | Use |
|--------|-----|
| According to the text ... | referring to the text |
| Can I ...? | asking permission |
| Congratulations! | celebrating success |
| Could you ...? | polite request |
| Do you mind if ...? | polite permission |
| For example, ... | giving examples |
| Good luck! | wishing luck |
| Have a good time! | wishing enjoyment |
| How about ...? | making a suggestion |
| I agree / I don't agree. | expressing agreement |
| I'd like ... | making a polite request |
| I'm afraid ... | politely giving bad news |
| I'm sorry to hear that. | showing sympathy |
| I think / I believe ... | expressing an opinion |
| In my opinion, ... | expressing an opinion |
| Let's ... | suggesting an activity |
| Nice to meet you! | meeting someone |
| On the other hand, ... | contrasting ideas |
| See you later! | farewell |
| That sounds great! | expressing enthusiasm |
| To sum up, ... | summarising |
| Well done! | praising |
| What about ...? | making a suggestion |
| What does ... mean? | asking about meaning |
| What time does ... start/finish? | asking about times |
| Would you like ...? | offering something |

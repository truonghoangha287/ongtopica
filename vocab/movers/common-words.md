## Conjunctions and Discourse Markers
| Word | Example |
|------|---------|
| after | She eats after she finishes her homework. |
| although | Although it rained, we had fun. |
| because | She is happy because it's her birthday. |
| before | Wash your hands before eating. |
| first | First, mix the flour and butter. |
| however | However, she was very tired. |
| if | If it rains, we stay inside. |
| next | Next, add the eggs. |
| so | It was cold, so she put on her coat. |
| then | Then, put it in the oven. |
| when | When I grow up, I want to be a teacher. |
| while | He reads while she paints. |

## Useful Phrases
| Phrase | Use |
|--------|-----|
| What does ... mean? | asking about meaning |
| How do you say ... in English? | asking for a word |
| Can I ...? | asking permission |
| Would you like ...? | offering something |
| I'd like ... | making a polite request |
| What about ...? | making a suggestion |
| Let's ... | suggesting an activity |
| I think ... | expressing an opinion |
| I agree / I don't agree. | agreement |
| That's right. | confirming |
| Well done! | praising |
| Good luck! | wishing luck |
| Congratulations! | celebrating |
| Have fun! | wishing enjoyment |
